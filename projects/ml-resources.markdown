# Machine Learning Resources
Here's a running list of resources I've been using. The main purpose is to keep them in a central location for myself though.

## Resources related to type theory and intuitionism
**Intuitionism**

1. [Mathematical Intuitionism: an introduction to proof theory](https://catalog.lib.uchicago.edu/vufind/Record/892339), *by Dragalin, Alʹbert Grigorʹevich.*
2. [Intuitionistic Logic](./files/resources/van-dalen-intuitionistic-logic.pdf), *by Dirk van Dalen.*

**Type Theory**

1. [Intuitionistic Type Theory](./files/resources/martin-lof-intuitionistic-type-theory.pdf), *by Pierre Martin-Löf.*
2. [Programming in Martin-Löf Type Theory](./files/resources/programming-in-martin-lof-type-theory.pdf), *by Bengt Nordström, Kent Petersson, and Jan M. Smith.*
3. [Lecture Notes on the Lambda Calculus](./files/resources/selinger-lecture-notes-on-the-lambda-calculus.pdf), *by Peter Selinger.*
4. [Introduction to Type Theory](./files/resources/geuvers-introduction-to-type-theory.pdf), *by Herman Geuvers.*
5. [Martin-Löf's Type Theory](./files/resources/nordstrom-martin-lofs-type-theory.pdf), *by Bengt Nordström, Kent Petersson, and Jan M. Smith.*
6. [Type Theory and Functional Programming](./files/resources/thompson-type-theory-and-functional-programming.pdf), *by Simon Thompson.*
7. [Type Theory and Formal Proof](https://www.amazon.com/Type-Theory-Formal-Proof-Introduction/dp/110703650X), *by Rob Nederpelt and Herman Geuvers.*

**Category Theory**

1. [Category Theory](./files/resources/awodey-category-theory.pdf), *by Steve Awodey.*
2. [From Sets to Types to Categories to Sets](./files/resources/awodey-sets-types-categories.pdf), *by Steve Awodey.*
3. [Category Theory for the Working Mathematician](./files/resources/maclane-category-theory-for-the-working-mathematician.pdf), *by Saunders Mac Lane.*
4. [Category Theory for Computer Scientists](./files/resources/category-theory-for-computer-scientists.pdf), *by Michael Barr and Charles Wells.*

**Homotopy Type Theory**

1. [Homotopy Type Theory](./files/resources/hott.pdf)
2. [Homotopy Type Theory Lectures](https://www.cs.cmu.edu/~rwh/courses/hott/), *by Robert Harper.*
3. [Groupoid Interpretation of Type Theory](./files/resources/groupoid-interpretation-of-type-theory.pdf), *by Martin Hofmann and Thomas Striecher.*
4. [The Path Model of Intensional Type Theory](./files/resources/ruch-the-path-model-of-intensional-type-theory.pdf), *by Fabian Ruch.*
5. [Homotopy-Initial Algebras in Type Theory](./files/resources/homotopy-initial-algebras-in-type-theory.pdf), *by Steve Awodey, Nicola Gambino, and Kristina Sojakova.*
6. [Homotopy Theoretic Models of Identity Types](./files/resources/homotopy-theoretic-models-of-identity-types.pdf), *by Steve Awodey and Michael Warren.*
7. [Type Theory and Homotopy](./files/resources/type-theory-and-homotopy.pdf), *by Steve Awodey.*

## Resources related to machine learning
**Statistical Machine Learning**

1. [Machine Learning: A Probabilistic Perspective](./files/resources/murphy-machine-learning-a-probabilistic-perspective.pdf), *by Kevin Murphy.*
2. [Deep Learning](https://github.com/HFTrader/DeepLearningBook), *by Ian Goodfellow, Yoshua Bengio, and Aaron Courville.*

## Resources related to physical/dynamical systems

1. [Quantum Theory: A Mathematical Approach](./files/resources/bongaarts-quantum-theory-a-mathematical-approach.pdf), *by Peter Bongaarts.*
2. [Introduction to the Modern Theory of Dynamical Systems](https://www.amazon.com/Introduction-Dynamical-Encyclopedia-Mathematics-Applications/dp/0521575575), *by Anatole Katok and Boris Hasselblatt.*
