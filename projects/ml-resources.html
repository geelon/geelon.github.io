Machine Learning Resources

Here’s a running list of resources I’ve been using. The main purpose is to keep them in a central location for myself though.

Resources related to type theory and intuitionism

Intuitionism

  1. Mathematical Intuitionism: an introduction to proof theory, by Dragalin, Alʹbert Grigorʹevich.
  2. Intuitionistic Logic, by Dirk van Dalen.

Type Theory

  1. Intuitionistic Type Theory, by Pierre Martin-Löf.
  2. Programming in Martin-Löf Type Theory, by Bengt Nordström, Kent Petersson, and Jan M. Smith.
  3. Lecture Notes on the Lambda Calculus, by Peter Selinger.
  4. Introduction to Type Theory, by Herman Geuvers.
  5. Martin-Löf’s Type Theory, by Bengt Nordström, Kent Petersson, and Jan M. Smith.
  6. Type Theory and Functional Programming, by Simon Thompson.
  7. Type Theory and Formal Proof, by Rob Nederpelt and Herman Geuvers.

Category Theory

  1. Category Theory, by Steve Awodey.
  2. From Sets to Types to Categories to Sets, by Steve Awodey.
  3. Category Theory for the Working Mathematician, by Saunders Mac Lane.
  4. Category Theory for Computer Scientists, by Michael Barr and Charles Wells.

Homotopy Type Theory

  1. Homotopy Type Theory
  2. Homotopy Type Theory Lectures, by Robert Harper.
  3. Groupoid Interpretation of Type Theory, by Martin Hofmann and Thomas Striecher.
  4. The Path Model of Intensional Type Theory, by Fabian Ruch.
  5. Homotopy-Initial Algebras in Type Theory, by Steve Awodey, Nicola Gambino, and Kristina Sojakova.
  6. Homotopy Theoretic Models of Identity Types, by Steve Awodey and Michael Warren.
  7. Type Theory and Homotopy, by Steve Awodey.

Resources related to machine learning

Statistical Machine Learning

  1. Machine Learning: A Probabilistic Perspective, by Kevin Murphy.
  2. Deep Learning, by Ian Goodfellow, Yoshua Bengio, and Aaron Courville.

Resources related to physical/dynamical systems

  1. Quantum Theory: A Mathematical Approach, by Peter Bongaarts.
  2. Introduction to the Modern Theory of Dynamical Systems, by Anatole Katok and Boris Hasselblatt.
